Make sure you have Node.js installed.

Install bower: npm install -g bower

Install gulp: npm install -g gulp

Navigate to root directory and install all node dependencies: npm install

Might take a couple of tries depending on the Node.js version used.

Navigate to root directory and install all bower dependencies: bower install

Run npm start to start a lite node server on port 80 and to compile source files in development mode. Any changes you do to the code will trigger a re-compilation.

Run gulp build --production to compile a production build with minification and obfuscation.

Try the authentication service using the mock credentials "user@auth" and "letmein" for the username and password respectively. Alternatively, you can use the mock sign up and gain access until you refresh the page (No information gets collected).
